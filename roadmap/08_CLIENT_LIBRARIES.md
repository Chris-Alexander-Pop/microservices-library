# Client Libraries & SDKs Roadmap

## Web (TypeScript / JavaScript)
- [ ] **React Hooks Library**:
    - `useAuth()` hook for the auth service
    - `useSocket()` for WebSocket service
- [ ] **Type Generation**:
    - `OpenAPI -> TypeScript` generation (Orval / RTK Query)
    - `gRPC -> TypeScript` generation (Buf Connect)

## Mobile
- [ ] **Flutter / Dart SDK**:
    - API Client wrapper
    - Auth State management
- [ ] **Swift (iOS) SDK**:
    - Combine / Async/Await network layer
- [ ] **Kotlin (Android) SDK**:
    - Retrofit / Coroutines integration

## Desktop / CLI
- [ ] **Go SDK**: A pre-built client library for interacting with your microservices from other Go apps.
- [ ] **Python SDK**: For data science / scripting integration.
